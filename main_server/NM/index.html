The following tool is made to Analyze the Network.


It consists of 4 modules which plots live data and even generate dataset


IP vs Duration Plot :- It simply generates IP and duration for each of them on network according to consumption of data


Current Devices :- This module plots all the paths taken by devices on the network and all the sub paths also


IPs on Map :- This module maps out all the external network on the webpage resolve them to location and puts them on a map


In-Out Bandwidth Curve:- This generated a plot with Input,Output,Total bandwidth consumption versus the Time taken by them. Red being IN,Blue being Total,Green being out
